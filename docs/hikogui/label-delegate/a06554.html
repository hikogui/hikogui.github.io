Detailed Description

A GUI widget that permits the user to make a binary choice.

A radio is a button with three different states with different visual representation:

  • on: A pip is shown inside the circle.
  • off: An empty circle is shown.

Each time a user activates the radio-button it toggles between the 'on' and 'off' states. If the radio is in the 'other' state an activation will switch it to the 'off' state.

In the following example we create a radio widget on the window which observes value. When the value is 1 the radio is 'on', when the value is 2 the radio is 'off'.
